Belajar CSS

Apa itu CSS

Css adalah singkatan dari Cascading Style Sheet yang dipergunakan untuk menentukan tampilan dan format pada sebuah website. Artinya, dengan adanya CSS bisa mengatur mulai dari font, warna tulisan dan latar belakang pada website tersebut.

Bagaimana menggunakan CSS?

CSS beroperasi dengan bahasa Inggris sederhana berbasis syntax dan dilengkapi sekumpulan perintah atau rule untuk mengaturnya. Salah satu rule CSS dapat beroperasi melalui tag style untuk atribut class warna. Dengan pengaturan CSS pada HTML ini akan mengubah warna teks menjadi lebih mudah.

  1. Inline Styles: Metode ini digunakan dengan cara memasukkan file css ke dalam tag HTML secara langsung.
  2. Internal : Metode ini menggunakan CSS dengan tag style pada file html. Dengan ini, kita bisa memisahkan antara tag html dengan kode css tetapi masih dalam satu file yang sama.
  3. Eksternal : Metode ini digunakan dengan cara memisahkan file html dan juga file css. Untuk memasukkan file css ke dalam html bisa menggunakan kode html seperti berikut.
